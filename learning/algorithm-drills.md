# Algorithm Drills

算法训练目标不是刷很多题，而是补齐 Python 基础数据结构和复杂度意识。

推荐目录：

```text
algorithms/
tests/test_algorithms.py
```

## Rules

- 每个算法一个函数。
- 每个函数至少 3 个测试。
- 每题写一句复杂度说明。
- 优先用 Python 内置结构：`list`、`dict`、`set`、`tuple`。

## Week Drill List

### Day A - List And Dict

任务：

- `two_sum(nums, target)`
- `count_items(items)`

重点：

- `dict` 查找
- 避免双重循环

### Day B - String

任务：

- `reverse_words(text)`
- `is_palindrome(text)`

重点：

- 字符串清洗
- 切片

### Day C - Stack

任务：

- `valid_parentheses(text)`

重点：

- 栈
- 映射表

### Day D - Queue

任务：

- `moving_average(nums, size)`

重点：

- 窗口
- 队列思路

### Day E - Group And Sort

任务：

- `group_by_weather(records)`
- `top_n(items, n)`

重点：

- 分组
- 排序 key

### Day F - Recursion

任务：

- `factorial(n)`
- `flatten(items)`

重点：

- 递归出口
- 输入边界

### Day G - Review

任务：

- 给所有函数补测试。
- 写每题复杂度。
- 找出最不熟的一类题。
